# ArtAscii
Simple ascii art generator

```
Usage: ArtAscii [options] (input image) [output file]
 Options:
  -c  (character set)   A set of characters to use for the mapping
  -lf                   List system fonts and exit
  -sf (font name)       Use system font with given name
  -f  (font file)       Font file used for rendering characters
  -fs (style)           Choose a font style
  -tw (number)          Width in characters of output
  -th (number)          Height in characters of output
  -it (text file)       A utf-8 encoded file to use as the source of characters
  -ot                   Output as text instead of an image

 Characer Sets:
  None
  Ascii
  CodePage437
```
